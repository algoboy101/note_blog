> Windows下的有道词典非常地好用，在Ubuntu下也可以使用有道词典啦！！！

## 添加源，并更新
```
# 添加软件源
sudo add-apt-repository ppa:justzx2011/openyoudao-v0.4
# 更新
sudo apt-get update
```
## 安装软件包
```
# 安装软件包
sudo apt-get install openyoudao
```
